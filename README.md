# juztix_joblisting

A simple and modern Job Center for **FiveM ESX** servers. This resource allows players to choose, change, or quit jobs easily while enhancing roleplay and server organization.

---

##  Features

* Easy job selection and job switching
* Clean and simple user interface
* ESX compatible
* Lightweight and performance-friendly
* Roleplay-friendly job management

---

## Requirements

* ESX Framework (Legacy or compatible version)
* es_extended

---

##  Installation

1. Download or clone this resource
2. Place the folder into your `resources` directory
3. Add the following line to your `server.cfg`:

```cfg
ensure juztix_joblisting
```

4. Restart your server

---

##  Usage

* Players can access the Job Center via the configured location or interaction
* Select a new job from the available list
* Quit or change jobs at any time (depending on configuration)

---

##  Configuration

You can configure the Job Center in the `config.lua` file:

* Job list
* Job Center location
* UI options
* Permissions (if enabled)

Make sure to restart the server after changing the configuration.
